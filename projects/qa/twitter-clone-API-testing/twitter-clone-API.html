Twitter-Clone

Twitter-Clone is a social media platform where the user can share his thoughts and opinion by tweeting. Users can see others’ tweets and discuss with them diverse topics, news and opinions through likes and replies.

Specifications

I examined the documentation of Twitter-Clone API. I then rewrote it in OpenAPI using Swagger. You can read the OpenAPI documentation using the Swagger UI or directly in my Github repository.

Test cases

I created 94 test cases for the 16 endpoints, first using Postman. I then automated them using Cypress. You can find all the automated test cases for Twitter-Clone in my Github repository.

🐞 Bug reports

My tests found 1 bug related to the API testing of the Twitter-Clone app. The bug can be transferred to the development team for fixing

  • Bug 1: Register using username with special characters other than ".", "-"and "_" gives an error of 500 instead of 400

    Summary

    When registering, username must contain numbers, letters and only ".", "-", "_" as special characters.
    If the user writes an invalid username, for example "johnny#", the error should be 400 instead of 500.

    Steps to reproduce
    1. Activate Twitter-Clone web application in order for the API testing to work
    2. Open Postman, which I used for the Twitter-Clone API testing, or any other tools that can be used for API testing
    3. Insert the following information: (1) POST, (2) url: "http://localhost:3001/api/auth/register", (3) body: { name: "Johnny", email: "johnny@gmail.com", username: "johnny#", password: "Clonejohnny23", }
    4. Click the button "Send"
    Actual result 🐞

    Since "#" is an invalid special character to be used in the username, the error is 500

    Expected result

    The error is 400
