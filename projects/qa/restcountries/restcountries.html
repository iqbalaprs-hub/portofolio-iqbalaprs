Rest Countries

Rest Countries is a web application that provides details about countries such as name, alpha codes, currencies, capital, region, subregion, language, translations... In order to access the information, this web application is showing you how to do it by using the GET API request (GET API request is a type of communication designed to request data and retrieve information from the server).

Specifications

I have examined the website application "Rest Countries" and wrote specifications (or documentation) for the GET API request using openAPI Swagger, which it have provided me a guide for writing test cases. You can find the documentation in my Github repository and you can see it tested on openAPI swagger.

Test cases

We created 66 test cases for 10 endpoints. You can find all the test cases for RestCountries in my Github repository. Even thought we did manual tests on Postman, we had to do the automated tests on Cypress.

Cypress tests

All test cases above where rewritten as automated tests using Cypress. You can find them in my Github repository. Here's a video capture of all test runs.
