Rest Countries

The Rest Countries Public API is a free online service that provides information about countries around the world. It offers data such as country names, capital cities, population, currency, languages, and more, accessible through simple HTTP requests for easy integration into various applications.
The objective of this project is to write API tests that evaluates the endpoints exposed by this API, and report bugs discovered during the tests.

OpenAPI.yaml - Cypress tests - Swagger UI - Test run

  • Specifications

    I examined the documentation of Rest Countries API. I then rewrote it in OpenAPI using Swagger. You can read the OpenAPI documentation using the Swagger UI or directly in my Github repository.

  • Test cases

    I created 66 test cases for the 10 endpoints, first using Postman. I then automated them using Cypress. You can find all the automated test cases for Rest Countries in my Github repository.
